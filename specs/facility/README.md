- `OFFICE.md`: family `7`
- `CONDO.md`: family `9`
- `COMMERCIAL.md`: families `6`, `12`, `10`

- `PARKING.md`: families `11` (parking space), `44` (parking ramp)
- `EVALUATION.md`: evaluation visitors and Tower advancement
